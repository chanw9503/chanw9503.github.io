# 두 수의 합

 
### 문제 설명

정수 num1과 num2가 주어질 때, num1에서 num2를 뺀 값을 return하도록 soltuion 함수를 완성해주세요.

---
### solution.js
function solution(num1, num2) {
    
    var answer = 0;
    return answer;
}

---

### 제한 사항
-50000 ≤ num1 ≤ 50000 

-50000 ≤ num2 ≤ 50000

---
### 입출력 예
| num1 | num2 | result |
|:---:|:----:|:----|
| 2 | 3 | -1 |
| 100 | 2 | 98 |

### 입출력 예 설명

### 입출력 예 #1
 - num1 이 2이고 num2가 3이므로 2 -3 = -1 을 return 합니다. 
 
 ### 입출력 예 #2
 - num1 이 100이고 num2가 2이므로 100 -2 = 98 을 return 합니다. 

---

### Result Code
```
function solution(num1, num2) {
    
    var answer = 0;
    answer = num1 - num2; 
    return answer;
}
```
