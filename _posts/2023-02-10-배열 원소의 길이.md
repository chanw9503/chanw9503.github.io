> filter 정의 

```
 arr.filter(callback(element[, index[, array]])[, thisArg])
```


> 매개변수

`callback`

각 요소를 시험할 함수. true를 반환하면 요소를 유지하고, false를 반환하면 버립니다. 

다음 세 가지 매개 변수를 받습니다.

`element`

처리할 현재 요소.

`index`

처리할 현재 요소의 인덱스

`array`

filter 를 호출한 배열

`thisArg`

callback을 실행할 때 this로 사용하는 값.


```
const team14 = [
    {name: 'YC' , age: 32 },
    {name: 'MJ' , age: 33},
    {name: 'cw' , age: 29},
    {name: 'DH' , age: 26}

const oldman = team14.filter(man => man.age > 30)

console.log(oldman);
```
